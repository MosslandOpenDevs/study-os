# MVP Scope

## Included

### Document ingestion
- PDF upload
- markdown/text upload
- chunking into study units
- source citation tracking

### Study generation
- Korean summary
- key concept extraction
- confusion point extraction
- quiz generation from selected sections

### Assessment
- multiple choice
- short answer
- fill-in-the-blank
- answer checking and explanation

### Error notebook
- save wrong answers
- reason tagging
- attach source concept
- queue for retry later

### Review
- daily review list
- spaced repetition scheduling
- overdue review surfacing

### Planning
- set target exam/goal
- D-day countdown
- weekly plan
- daily task breakdown

## Excluded
- social classroom features
- school administration features
- advanced parent dashboards
- voice tutoring
- live lecture assistance

## MVP user journey
1. User uploads material
2. System extracts study units
3. User gets Korean summary and generated quiz
4. User solves quiz
5. Wrong answers go into error notebook
6. Scheduler adds review tasks
7. Dashboard shows progress toward target goal
