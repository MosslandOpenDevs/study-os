# User Personas

## 1. University student
Needs:
- summarize dense lecture material
- generate likely exam questions
- review weak concepts quickly

Pain points:
- too much material
- inconsistent review habits
- poor visibility into weak topics

## 2. Certificate exam learner
Needs:
- structured daily plan
- repeated practice on weak areas
- tight D-day management

Pain points:
- manual planning overhead
- forgetting wrong answers
- unclear progress over time

## 3. Developer learning from English docs
Needs:
- Korean explanation of English technical content
- concept quizzes from docs
- fast review of important details

Pain points:
- translating mentally while studying
- hard to retain details from long docs
- no clean workflow from docs to practice

## 4. Tutor or teacher
Needs:
- generate practice material from source docs
- assign and review mistakes quickly
- identify common weak points

Pain points:
- repetitive worksheet creation
- fragmented note and quiz workflow
- slow feedback loop
