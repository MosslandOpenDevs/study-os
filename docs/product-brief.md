# Product Brief

## Product name
study-os

## One-line description
A Korean-first AI study operating system that turns learning materials into summaries, quizzes, review loops, and exam plans.

## Problem
Most AI learning tools are good at answering questions, but not at managing the full study workflow. Korean learners often need a more structured loop:
- organize source materials
- summarize them clearly in Korean
- practice with generated questions
- keep an error notebook
- revisit weak concepts on a schedule
- prepare toward a target exam date

Without that workflow, learners end up stitching together chat tools, note apps, PDF viewers, and manual review systems.

## Vision
Build a study operating system for Korean learners where uploaded materials become a continuous loop of learning, testing, review, and progress tracking.

## Success criteria for MVP
- a learner can upload one document set
- generate a Korean summary and quiz set from it
- answer questions and store mistakes
- receive a daily review queue
- see progress toward a target study goal

## Primary use cases
1. Exam prep from lecture PDFs
2. Converting English technical material into Korean study assets
3. Building a personalized error notebook from practice sessions
4. Planning study by D-day instead of ad hoc chat sessions

## Product principles
- source-grounded outputs
- Korean-first UX
- measurable study progress
- minimal friction from material to review
- structured learning over generic chat
