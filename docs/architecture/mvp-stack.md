# MVP Tech Stack Proposal

## Frontend
- Next.js
- TypeScript
- Tailwind CSS

## Backend
- Node.js
- TypeScript
- modular service structure

## Data
- Postgres for core application data
- object storage for uploaded files
- optional pgvector or separate vector layer later

## AI integration
- provider abstraction layer
- Korean summary prompt set
- quiz generation prompt set
- citation-aware answer generation

## Why this stack
- strong hiring and contributor familiarity
- fast iteration speed
- easy monorepo structure
- good fit for dashboard + API + worker model

## Delay until later
- native apps
- heavy microservice split
- advanced multi-model routing
- real-time collaboration infra
